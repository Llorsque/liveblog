⚙ GitHub-instellingen

Met een Personal Access Token kun je posts direct vanuit deze pagina publiceren — ook vanaf je telefoon. De token wordt alleen in deze browser opgeslagen (localStorage), niet ergens online.

  1. Ga naar github.com/settings/personal-access-tokens (fine-grained).
  2. Repository access → Only select repositories → kies je liveblog-repo.
  3. Permissions → Repository permissions → Contents: Read and write.
  4. Genereer en plak de token hieronder.
Wordt gebruikt om bestaande posts op te halen zonder token.
